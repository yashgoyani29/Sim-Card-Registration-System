SIM CARD MANAGEMENT SYSTEM - JAVA
================================

PROJECT DESCRIPTION
-------------------
This is a console-based Java application that simulates
a basic SIM Card Management System. The application allows
users to issue a SIM card by entering personal details and
view the stored information. The project is menu-driven
and demonstrates core Java and OOP concepts.

---------------------------------------------------------
FEATURES
---------------------------------------------------------
1. Menu-driven console interface
2. Issue a new SIM card
3. Accept user details:
   - Name
   - Address
   - Aadhaar Number
4. Display SIM card information
5. Simple and beginner-friendly logic
6. Runs entirely in Command Prompt / Terminal
7. Demonstrates core Java concepts

---------------------------------------------------------
TECHNOLOGIES USED
---------------------------------------------------------
- Java (Core Java)
- Object-Oriented Programming (OOP)
- Scanner class
- Command Line Interface (CLI)

---------------------------------------------------------
FOLDER STRUCTURE
---------------------------------------------------------
```txt
Project_Folder/
|
|-- Sim.java        -> Java source file (SIM logic)

```
NOTE:
The source file Main.java is not available.
Only the compiled Main.class file is present.

---------------------------------------------------------
PREREQUISITES
---------------------------------------------------------
1. Java JDK (Version 8 or above)
2. Command Prompt / Terminal
3. Basic knowledge of Java

Check Java installation:
```bash
java -version
```
---------------------------------------------------------
HOW TO RUN THE PROJECT
---------------------------------------------------------

METHOD 1: RUN USING EXISTING .CLASS FILES
-----------------------------------------
1. Open Command Prompt / Terminal
2. Navigate to the project directory
3. Run the command:
```txt
   java Main
```
---------------------------------------------------------
METHOD 2: COMPILE AND RUN (IF Main.java IS AVAILABLE)
---------------------------------------------------------
1. Compile the source files:

   javac Sim.java Main.java

2. Run the program:
```bash
   java Main
```
---------------------------------------------------------
APPLICATION FLOW
---------------------------------------------------------
1. Program starts
2. Menu is displayed
3. User selects an option
4. User enters required SIM details
5. Data is stored and displayed
6. User exits the application
